# 🚀 Photogram Backend

Серверная часть веб-приложения «Фотограм» на Spring Boot.

## 🛠 Стек технологий

| Компонент | Технология | Версия | Назначение |
|-----------|------------|--------|------------|
| **Язык программирования** | Java | 21 | Основной язык разработки |
| **Фреймворк** | Spring Boot | 3.2.0 | Быстрое создание Spring-приложений |
| **Веб-модуль** | Spring Web | 6.1.1 | Обработка HTTP-запросов, REST контроллеры |
| **Безопасность** | Spring Security | 6.2.0 | Аутентификация и авторизация |
| **ORM** | Spring Data JPA | 3.2.0 | Работа с базой данных через сущности |
| **JPA-провайдер** | Hibernate | 6.3.1 | Маппинг объектов на таблицы БД |
| **База данных (локально)** | H2 Database | 2.2.x | Встроенная БД для разработки |
| **Утилиты кода** | Lombok | 1.18.30 | Генерация геттеров, сеттеров, конструкторов |
| **JWT-токены** | jjwt (Java JWT) | 0.12.3 | Генерация и валидация токенов |
| **Хеширование** | BCrypt | (в Spring Security) | Безопасное хранение паролей |
| **Сборка проекта** | Apache Maven | 3.9.x | Управление зависимостями и сборка |
| IDE | IntelliJ IDEA |
