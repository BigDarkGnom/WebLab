# 📸 Photogram Frontend

Клиентская часть веб-приложения «Фотограм» на React.

## 🛠 Стек технологий

| Компонент | Технология | Версия | Назначение |
|-----------|------------|--------|------------|
| **Язык** | JavaScript (ES6+) | — | Основной язык разработки |
| **Библиотека UI** | React | 18.2+ | Построение пользовательского интерфейса |
| **Сборщик** | Vite | 5.0.x | Быстрая сборка и hot-reload |
| **HTTP-клиент** | Axios | 1.6.x | Отправка запросов к бекенду |
| **Роутинг** | React Router DOM | 6.20.x | Навигация между страницами |
| **CSS-фреймворк** | Bootstrap | 5.3.x | Стилизация компонентов |
| **React-компоненты** | React Bootstrap | 2.9.x | Готовые UI-компоненты для React |
| **Пакетный менеджер** | npm | 10.x | Установка зависимостей |
|IDE | Node.js |
